How to use this blueprint

Drag any node to reposition it — layout saves automatically in your browser.

Double-click a title or subtitle to edit it inline. Press Enter or click away to save.

Hover a node and click ✓ Done to mark it complete. Click Change state to cycle through states.

Right-click any node to change its state, edit text, or delete it.

Export PNG saves the full canvas as a high-res image file.

↺ Positions resets layout only. ✕ Reset all clears everything including edits and done states.
